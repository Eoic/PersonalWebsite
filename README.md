# Personal website

## Development

### Dependencies

- Install NodeJS dependencies:

```bash
npm ci
```

- Install Python dependences:

```bash
python -m venv .venv
. .venv/bin/activate

```

#### Scripts

- Start development server:

    ```bash
    npm run dev
    ```

- Build site for production:

    ```bash
    npm run build
    ```

- Serve production site:

    ```bash
    npm run serve
    ```
