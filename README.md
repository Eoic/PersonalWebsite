# Personal website

### Development

#### Dependencies

- Install NodeJS dependencies:

```
npm ci
```

- Install Python dependences:

```
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

#### Scripts

- Start development server:

    ```bash
    npm run dev
    ```

- Build site for production:

    ```bash
    npm run build
    ```

- Serve production site:
    ```bash
    npm run serve
    ```
